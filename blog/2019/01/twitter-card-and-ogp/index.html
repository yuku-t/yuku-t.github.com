Twitter Card と OGP に対応した

  • 2019-01-07 13:40
  • ブログ開発記

Commit: yuku/yuku.github.io@5fe34e4

Open Graph protocol (OGP) と Twitter Card に対応して Twitter や Facebook でブログ記事の URL が共有されたときにリッチスニペットが表示されるようにしました。

ブログ書いてみたhttps://t.co/Gehy21W69V

— ゆうく (@yuku_t) 2019年1月7日

開発時はそれぞれのリンク先のドキュメントを参照しつつ実装し、 Facebook と Twitter が用意しているデバッガーを使って動作確認をします:

  • Card Validator | Twitter Developers
  • シェアデバッガー - 開発者向け Facebook

Facebook の仕様として fb:app_id が求められたので、新しく本ブログ用のプロジェクトを Facebook に作り、その ID を登録しました。 Web 上の古い情報によると昔は fb:admins などもサポートされていましたが、現在は fb:app_id に一本化されたようです。
